NAME
==================================================

**rush add** - Register a local repository.

SYNOPSIS
==================================================

**rush add** REPO PATH

DESCRIPTION
==================================================

Register a local repository.

This command adds the specified path to the configuration file.

- Alias: **a**

ARGUMENTS
==================================================

REPO
--------------------------------------------------

Repository name.

- *Required*

PATH
--------------------------------------------------

Path to the repository.

- *Required*

EXAMPLES
==================================================

~~~
rush add default ~/rush-repos/default

~~~
